normalize

join
拼接路径,也会规范化路径
resolve
一个相对路径的绝对路径
basename、dirname、extname
parse、format
1. 方法会从一个对象返回一个路径字符串。 与 path.parse() 相反。
2.  方法返回一个对象，对象的属性表示 path 的元素

root
base
name
ext

sep、delimiter、win32、posix
1. sep path路径的分隔符

2. delimiter process.env.path上的分隔符

3. win32属性提供了 path 方法针对 Windows 的实现
4.posix 属性提供了 path 方法针对 Windows 的实现
